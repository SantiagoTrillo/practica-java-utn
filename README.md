# Prácticas de Java - Programación II (UTN FRA)

Repositorio que reúne ejercicios, trabajos prácticos y prácticas de parcial desarrollados durante la cursada de la materia Programación II de la UTN Facultad Regional Avellaneda.

Los proyectos están organizados por temática y cada ejercicio es independiente.

## Estructura del repositorio

El repositorio se divide por ejes conceptuales:

- `poo/` – Programación orientada a objetos  
- `arreglos/` – Vectores y matrices  
- `colecciones/` – Uso de colecciones  
- `polimorfismo/` – Herencia, interfaces y polimorfismo  
- `parciales/` – Prácticas y parciales completos

Cada ejercicio suele incluir:
- Código fuente en `src/`
- Archivo `build.xml` y `manifest.mf`
- Diagrama UML (`uml.pdf`)
- Consignas en PDF
